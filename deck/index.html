Rules, Not Vibes

Teach the machine the brand. No vibes — only rules.

Typography

  • Archivo Black, all caps, centered
  • One scale — Display 64 / Heading 32 / Body 16
  • Tight tracking; never break a word

Color

  • Bone #EFEDEA + true black #000
  • Black ↔ bone inversion is the contrast rhythm
  • Accent sparingly

Spacing

  • 8px baseline grid — multiples of 8
  • Generous negative space

Motion

  • Weighty line-rise on cubic-bezier(.82,0,.18,1)
  • One element owns the motion
  • Motion is content — runs even when reduced

Layout

  • Fit every frame to 16:9 — type bleeds edge-to-edge
  • One idea per station

Voice

  • Plain, declarative, confident
  • No hype words ("premium", "seamless")
